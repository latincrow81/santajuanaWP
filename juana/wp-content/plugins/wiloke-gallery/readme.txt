== Description ==
Wiloke Gallery is a WordPress plugin, which was built to help you building nice looking galleries using images pulled from various sources like Flickr, Instagram, Picasa event Hosted-Images.

<h3><strong>Key Features & Options</strong></h3>
<ul>
<li>Beautiful Gallery</li>
<li>Lightbox</li>
<li>Get the image from flicrk ID</li>
<li>Get the image from Instagram user id</li>
<li>Get the image from Instagram tag</li>
<li>Get the image from Instagram popular</li>
<li>Free supports.</li>
</ul>
